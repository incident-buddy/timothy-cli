⚡ Self-Hosted CLI Tool

Timothy

Upload LLM-generated HTML straight from your terminal.
Share via a time-limited URL — no login required for viewers.

$ llm generate report | tim upload --stdin → https://your-api/s/01JWXYZ...

📌 Key point: Files are stored privately in Cloud Storage — never publicly accessible. All view requests are proxied through the API. Access expires after the specified TTL. Firestore tracks metadata (title, expiry, user ID).

Features

📤

File / stdin upload

Upload an HTML file directly, or pipe LLM output straight in

⏱️

Time-limited URLs

Default 7-day TTL. Use --ttl to set any number of days

📋

List & Delete

Manage uploads with tim list and tim delete

🔐

Self-hosted

You own the storage and access control — fully on your infra

🌐

IP restriction (optional)

Lock shared URLs to specific IPs or CIDRs via ALLOWED_IPS

CLI Demo

bash — timothy
# 1. One-time setup
$ tim setup
API key: ··················
API endpoint: https://your-api.example.com
# 2. Pipe LLM output and share
$ claude -p "write an analysis report" \
| tim upload --stdin --title "Analysis" --ttl 30
✓ https://your-api/s/01JWXYZ...
# 3. List uploads
$ tim list
ID TITLE CREATED EXPIRES
01JWXYZ... Analysis 2026-05-20 2026-06-19
# 4. Delete
$ tim delete 01JWXYZ...
Delete 01JWXYZ...? [y/N] y
✓ Deleted 01JWXYZ...

Deployment Options

identical codebase

☁️ Google Cloud Run recommended

Deploy a container to Cloud Run. Cold-start possible, but low cost.

Compute Cloud Run (min 0 / max 2 instances)
Storage Firebase Cloud Storage (private bucket)
DB Firestore (metadata management)
Secret Secret Manager (credentials)

🟠 AWS Lambda

Keep the Firebase backend, swap only the compute layer for AWS.

Compute Lambda (container image + Function URL)
Storage Firebase Cloud Storage (shared)
DB Firebase Firestore (shared with Cloud Run)
Secret AWS Secrets Manager (credentials)

Security Model

🔑 Upload / Manage Bearer API key auth (issued by server admin)
🔗 View shared URL Anyone with the link — no login required
🚫 Direct storage access Blocked — Cloud Storage is never public
🌍 IP restriction CIDR allowlist via ALLOWED_IPS (optional)
⏰ Expiry Access automatically revoked after TTL
📜 Firestore rules Direct client writes are denied

Installation

$ npm install -g timothy-cli

Run without installing
